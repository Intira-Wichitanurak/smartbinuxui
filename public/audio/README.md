# Audio Files For Hippo Voice

Place your cute hippo voice MP3 files in this folder using these exact names:

1. ready.mp3
2. capturing.mp3
3. analyzing.mp3
4. analyzed.mp3
5. recyclable.mp3
6. hazardous.mp3
7. organic.mp3
8. general.mp3
9. food_residue.mp3
10. unknown.mp3
11. thank_you.mp3

Character voice folders (recommended):

- /audio/female/ready.mp3 ... /audio/female/thank_you.mp3
- /audio/male/ready.mp3 ... /audio/male/thank_you.mp3

The app will try character folder first, then fallback to /audio root.

Notes:
- The app tries to play these files first.
- If a file is missing, it falls back to browser TTS automatically.
- Recommended format: mono MP3, 44.1kHz, 96-128 kbps, 1-4 seconds per line.
